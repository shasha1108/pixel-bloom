## § STEP 1 — 画幅、音效与场景模式

1. **画幅**：根据场景推荐比例，保留用户选择权：
   - 赛博宠物 / 像素盆栽 → 推荐 **3:4**（竖直构图，适合单主体 + 玻璃容器）
   - 像素养鱼 / 水族箱 → 推荐 **16:9**（横向游动空间）
   - 像素森林 / 花园 → 推荐 **4:3**（稳定观感）
   - 沉浸 / 冥想 / Ganzfeld → 推荐 **全视口**
   - 用户可自选：3:4 / 9:16 / 1:1 / 4:3 / 16:9 / 全视口
   - 固定比例用 CSS `aspect-ratio` 锁定；全视口用 `100vw × 100vh`
2. 询问是否包含音效（yes / no），含音效时后续需读 `references/audio-engine.md`
3. 确定场景模式（决定后续所有技术决策）：

| 场景模式 | 画布 | 特征 |
|---------|------|------|
| 封闭容器（穹顶/球体/玻璃瓶） | 固定画布（CSS aspect-ratio 锁定） | Z-index 三明治 + `transform:scale` 适配屏幕 |
| 开放场景（全视口天空/水下） | 全视口 | 无玻璃壳，CSS 极光光斑漂浮 |
| Ganzfeld（已触发） | 全视口 | 高明度 Aero 光场 + Skyspace 透镜容器 |

3.5 确定场景类型，按维度分解（防止元素拼贴——每个场景都是一个有纵深的世界，不是元素的平铺）：

| 场景类型 | 场景模式 | 维度分解（L0→L3，远→近绘制） |
|---------|---------|---------------------------|
| **像素风景** | 开放场景 | L0 天空+极远山脊（去饱和蓝移）→ L1 中远地形+小号主体 → L2 近景草地+中号主体+尺度参照 → L3 画面底框/窗台。**详见 `landscape-composition.md`** |
| **像素森林** | 开放场景 | L0 天空+远山（蓝灰去饱和）→ L1 中远树冠层（小尺寸+浅色+去饱和 35%）→ L2 近景树干+灌木+蘑菇+小径（正常色）→ L3 地面落叶+前景花草+光斑。**引导线**：小径从 L2 延伸到 L0 并收窄 |
| **像素花园** | 开放/封闭均可 | L0 天空/远景栅栏（去饱和 25%）→ L1 远景花丛（小尺寸+浅色）→ L2 近景主花+大叶+茎干（正常色+正常尺寸）→ L3 花盆边缘/土壤/前景碎石。**尺度参照**：L2 花旁的瓢虫/蝴蝶 |
| **像素养鱼** | 封闭容器 | L0 远处缸壁+背景水草（蓝绿去饱和 30%+模糊）→ L1 中层鱼群+沉木+中景石（中饱和）→ L2 近景主鱼+前景水草（正常色+清晰轮廓）→ L3 缸底沙砾+前景玻璃反射高光。**深度氛围**：L0→L2 蓝色递增衰减（模拟水体吸收） |
| **赛博宠物** | 封闭容器 | L0 背景环境（毛玻璃模糊+去饱和 40%）→ L1 宠物身体+主体（正常色+清晰）→ L2 前景细节（爪子/耳朵/尾巴尖端+微放大）→ L3 UI 面板（食盆/状态指示/互动热区）。**尺度参照**：L3 的食盆大小 → 宠物有多大 |
| **像素盆栽** | 封闭容器 | L0 背景窗台/桌面（毛玻璃模糊+去饱和 35%）→ L1 花盆+土壤表面（正常色）→ L2 主茎+主叶片（清晰轮廓+模型 A 堆叠摇摆）→ L3 前景花朵/新芽/水珠/光斑。**活的细节**：新芽缓慢生长（`scale` 从 0→1 在 60 秒内） |

**每层必须回答三个问题**（从 `landscape-composition.md §一` 继承）：

> 1. 这一层有什么**主体元素**？（大小、数量、颜色）
> 2. 这一层有什么**引导线**把视线拉到下一层？（路/茎/水草走向/容器边框）
> 3. 这一层有什么**细节**让观众发现？（生物/光斑/水珠/落叶）

**如果某一层三个问题都答不上来 → 这一层是空壳，删掉它。**

## § STEP 2 — 像素拆解

将用户描述逐层分解为像素基础元素 + 程序化模型：

- 主物体：有自主运动（鱼/宠物）→ FSM 三态 + Perlin noise；无自主运动（植物/摆件）→ React-only
- 副物体：至少选 2 种 A-D 程序化模型（如 A水草 + C珊瑚礁）
- 气泡必须用 `drawAeroBubble()`（偏移高光正圆），禁用 `rect()` 圆角替代
- 调色板从 `assets/palettes.json` 选取（6 套命名色板），禁止对单个物体随机 RGB

## § STEP 3 — 技术决策

5 项决策，逐一明确后才写代码：

| 决策项 | 选项 |
|--------|------|
| 画布类型 | 固定画布（CSS aspect-ratio 锁定比例）/ 全视口 |
| 运动模式 | FSM Wander/Chase/Flee / React-only |
| 玻璃类型 | 穹顶 / 球体 / Skyspace 透镜（Ganzfeld）/ 无 |
| 粒子 | Perlin 漩涡 / 下落 / 拖影残像（Ganzfeld：`fill(10,8,20,12)` 替代 `clear()`）/ 无 |
| 音效 | 交互 chime（默认）/ 174Hz 三角波底噪 / 432+216Hz 颂钵（Ganzfeld）/ 无 |
