---

## 一、色彩哲学

Frutiger Aero = Windows Vista / iOS 3-6 / Nintendo Wii 时代的**高饱和通透自然美学**。核心原则：

- 不是灰度莫兰迪、不是灰冷工业色、不是暗沉墨绿色
- 是"阳光照射下的清澈水体与植物"——科技乐观主义
- 必须有纯白 `screen` 抛光条带——没有高光白 = 没有灵魂

## 四、标准渐变公式（禁止双色平淡渐变，必须 3-Stop+）

### 公式 A：水体/玻璃通透渐变

```javascript
let grad = ctx.createLinearGradient(x1, y1, x2, y2);
grad.addColorStop(0,   '#00F0FF'); // 顶部：透亮水青
grad.addColorStop(0.55,'#0090E0'); // 中段：清澈海蓝
grad.addColorStop(1,   '#05254D'); // 底部：深海靛蓝
```

### 公式 B：生态草地/植物渐变

```javascript
grad.addColorStop(0,   '#D0FF60'); // 顶部：阳光黄绿
grad.addColorStop(0.4, '#80E040'); // 中段：嫩芽绿
grad.addColorStop(1,   '#40B010'); // 底部：生态深绿
```

### 公式 C：亚克力抛光盖板（必须 screen 混合）

```javascript
let specGrad = ctx.createLinearGradient(x, yTop, x, yBottom);
specGrad.addColorStop(0,   'rgba(255,255,255,0.95)');
specGrad.addColorStop(0.35,'rgba(255,255,255,0.40)');
specGrad.addColorStop(0.65,'rgba(200,245,255,0.10)');
specGrad.addColorStop(1,   'rgba(255,255,255,0.0)');
ctx.globalCompositeOperation = 'screen';
```

## 五、色彩校验铁律

1. **HSB/HSV 下限**：除 `#05254D` 外，主色调明度(B) > 75%；饱和度(S) 在 60%-90%，严禁 < 30%
2. **3:1:1 面积比**：70% 冷色 + 20% 绿/白 + 10% 暖光点缀
3. **高光必须 screen/lighter**：禁止直接用不透明白块平涂
4. **渐变必须 3-Stop+**：禁止双色平淡渐变
5. **禁止自创颜色**：触发此文件后，所有 hex/RGBA 必须来自上述色板——凭直觉选色 = 必然出错
